  oc get route {{ include "circuit-breaker.fullname" . }}-frontend \
    -n {{ include "circuit-breaker.namespace" . }} \

Then open http://localhost:8080 in your browser.

VERIFYING THE CIRCUIT BREAKER
──────────────────────────────
1. Confirm that both Pods are running and that Consul has injected the Envoy
   sidecars:

     oc get pods -n {{ include "circuit-breaker.namespace" . }}
     # Each Pod should have 2/2 (or 3/3 if init-containers are counted)
     # containers in the READY column.

2. Check the Consul service mesh CRD statuses:

     oc get servicedefaults,serviceintentions,proxydefaults \
       -n {{ include "circuit-breaker.namespace" . }}
     # All resources should show synced=True.

3. Trigger errors on the backend to trip the circuit breaker.  Set the
   ERROR_RATE env-var to "1.0" (100 % errors) and watch Envoy eject the host:

     oc set env deployment/{{ include "circuit-breaker.fullname" . }}-backend \
       ERROR_RATE=1.0 \
       -n {{ include "circuit-breaker.namespace" . }}

4. After {{ .Values.circuitBreaker.passiveHealthCheck.consecutive5xx }} consecutive
   5xx responses the host is ejected for
   {{ .Values.circuitBreaker.passiveHealthCheck.baseEjectionTime }}.  Restore
   normal behaviour with:

     oc set env deployment/{{ include "circuit-breaker.fullname" . }}-backend \
       ERROR_RATE=0 \
       -n {{ include "circuit-breaker.namespace" . }}

BLUE/GREEN BACKEND SPLIT
─────────────────────────
Blue/green traffic splitting is ENABLED.  Consul is routing backend traffic
with a {{ .Values.blueGreen.trafficSplit.v1 }}/{{ .Values.blueGreen.trafficSplit.v2 }} split:

  v1 ({{ .Values.blueGreen.backendVersions.v1.name }}) → {{ .Values.blueGreen.trafficSplit.v1 }}%  ({{ .Values.blueGreen.backendVersions.v1.replicaCount }} replica(s))
  v2 ({{ .Values.blueGreen.backendVersions.v2.name }}) → {{ .Values.blueGreen.trafficSplit.v2 }}%  ({{ .Values.blueGreen.backendVersions.v2.replicaCount }} replica(s))

Verify the split with:

  kubectl get serviceresolvers,servicesplitters -n {{ include "circuit-breaker.namespace" . }}

To shift all traffic to v2 (full cut-over):

  helm upgrade {{ .Release.Name }} ./helm/circuit-breaker \
    --reuse-values \
    --set blueGreen.trafficSplit.v1=0 \
    --set blueGreen.trafficSplit.v2=100
